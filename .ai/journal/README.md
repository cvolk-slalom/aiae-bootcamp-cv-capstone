# Journal

Dated session log. One file per day: `YYYY-MM-DD.md`. Append sections within a day.

**Purpose:** record how AI accelerated this build, not just what changed. Every entry calls out which scaffold piece (context routing, briefs, ADRs, workflows, automation) carried the load. Git already has the diffs — this is the meta-trail.

Format and rules: see [.ai/workflows/update-journal.md](../workflows/update-journal.md).

The journal is for **humans skimming history**, **AI agents picking up cold**, and **future demos** of how an AI-first SDLC compounds over time. Keep entries terse.
